# Pulse: AI Operating System for FIFA World Cup 2026

Pulse is a world-class, production-ready AI Operating System designed to handle Navigation, Crowd Management, Accessibility, Transportation, Sustainability, and Emergency Response for the FIFA World Cup 2026.

## Architecture
Pulse operates as a unified platform:
- **Frontend**: Next.js (App Router), React, Tailwind CSS, Framer Motion.
- **Backend**: FastAPI (Python) for ultra-fast API delivery and AI orchestration.
- **Database/Realtime**: Supabase (PostgreSQL).
- **AI Engine**: Google Gemini 2.5 Flash via Vertex AI.

## Directory Structure
- `/frontend`: Next.js web application.
- `/backend`: FastAPI Python server.
- `/docs`: Comprehensive system documentation.
- `/scripts`: Utility scripts for deployment and mock data generation.

Please refer to the `/docs` directory for detailed environment setup and development guides.
